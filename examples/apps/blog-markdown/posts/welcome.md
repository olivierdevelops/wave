# Welcome to the Wave blog

This is the first post on a blog whose **entire infrastructure** is one YAML
file and a folder of `.md` files. No build step, no static-site generator,
no JavaScript framework — just Wave reading markdown off disk and rendering
it with Prism syntax highlighting.

## What we'll cover

- How the markdown route works
- Why we picked Wave over Hugo / Jekyll
- Tips for embedding code, tables, and images

Stay tuned.
